Sett verdi i documentet

En kan bruke httpbin for å teste HTTP-forespørsler. Om en henter https://httpbin.org/get får en tilbake JSON. Objektet har en nøkkel, headers, som inneholder informasjon som nettleseren sendte.

Oppgave: Hent https://httpbin.org/get og lagre headers fra responsen i tekstfeltet under.

Hint: JSON.stringify(headers, null, 2) vil gjøre objektet headers om til tekst.

async function test (input) {
    addScriptInIframe(input)
    await sleep(500)

}

function sleep (ms) {
    return new Promise((resolve) => {
        setTimeout(resolve, ms)
    })
}
